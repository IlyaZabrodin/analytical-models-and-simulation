## Задача 1 — Эргодическая цепь Маркова (`dz1/`)

Анализ цепи Маркова, заданной стохастической матрицей переходов (вариант 69).

**Задание:**
1. Нарисовать граф цепи.
2. Проверить выполнение критерия эргодичности.
3. Рассчитать предельные вероятности.
4. Записать предельную матрицу переходов.
5. Провести имитационное моделирование:
   - Случайно выбрать начальное состояние.
   - Случайно разыграть переход в новое состояние по распределению вероятностей.
   - Совершить 100 переходов; повторить эксперимент 50 раз.
   - Построить графики переключений состояний для 3 произвольных экспериментов.
   - Составить таблицу относительных частот вхождения в каждое состояние.
   - Рассчитать выборочные средние и исправленные СКО относительных частот.

## Задача 2 — Цепь Маркова с классами состояний (`dz2/`)

Анализ цепи Маркова с существенными и несущественными состояниями.

**Задание:**
1. Нарисовать граф цепи.
2. Выделить классы существенных и несущественных состояний.
3. Перенумеровать состояния так, чтобы смежные номера соответствовали каждому классу.
4. Рассчитать предельные распределения вероятностей для всех классов существенных состояний.
5. Сформировать предельную стохастическую матрицу без матричных операций над полной матрицей.
6. Провести имитационное моделирование:
   - Перебрать все состояния в качестве исходных.
   - Совершить 100 переходов; повторить 10 раз для каждого исходного состояния.
   - Построить графики переключений: по 2 старта внутри каждого класса существенных состояний, 6 стартов внутри класса несущественных.

## Прикладная задача — Прогноз температуры (`weather/`)

Моделирование температурных изменений с помощью дискретных цепей Маркова (ДЦМ).

**Постановка задачи:** по данным [rp5.ru](https://rp5.ru) рассчитать вектор вероятностей распределения температуры на 1-й декаде мая π(10), зная начальное состояние на 1-й декаде февраля π(1). Температурная шкала дискретизирована на N диапазонов.

**Три варианта решения:**
- **Вариант 1:** Единая матрица перехода между 1-й декадой февраля и 1-й декадой мая: π(10) = P^T(1,10) × π(1)
- **Вариант 2:** Однородная цепь Маркова — одна матрица для всех соседних декад: π(10) = (P^T)^9 × π(1)
- **Вариант 3:** Неоднородная цепь — отдельные матрицы для каждой пары соседних декад: π(10) = P^T(9,10) × … × P^T(1,2) × π(1)
